Powershell Live Response Artifacts Collector engine is written purely in Powershell which is shipped by default in every Windows machine since Windows 7

The colection scripts are picked by analysts from the graphical user interface artifacts list, and based on the particular artifacts analysis needs

While there is a multiple tools out there to perform Live Response, this particular tool focuses on Powershell, and its built in providers capabilities only, and without any agents installed previously on the requested machine
